# Trading Research (Sharp Research Course)

This repository contains my notes, notebooks, and code written while following the **Sharp Research trading course** on YouTube.

Playlist link:  
https://www.youtube.com/playlist?list=PLMyHgTuG7G4_LtKvFC3STetUiJ3iw3Vdh

---

## Purpose
The goal of this project is to build foundational skills in:
- Financial data collection and cleaning
- Technical indicators and strategy logic
- Backtesting and performance evaluation
- Research workflows using Python (pandas, NumPy, matplotlib, etc.)

---

## Repository Structure
- `*.ipynb` — Jupyter notebooks from each lesson / experiment  
- `data/` — datasets (if applicable)  
- `src/` — helper scripts and reusable strategy code (if applicable)

## Status
This repo is actively updated as I progress through the course.  
Work may be incomplete or change as strategies and experiments evolve.
